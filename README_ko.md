## 면책 조항

OpenList는 OpenList 팀이 독립적으로 유지 관리하는 오픈소스 프로젝트로, AGPL-3.0 라이선스를 따르며 완전한 코드 공개와 수정 투명성을 유지하는 것을 원칙으로 합니다.

커뮤니티 내에서 OpenListApp/OpenListApp 등 이 프로젝트와 유사한 이름을 사용하는 서드파티 프로젝트나, 동일하거나 유사한 명칭을 사용하는 유료 독점 소프트웨어가 등장하고 있습니다. 사용자의 혼란을 방지하기 위해 다음을 명확히 밝힙니다:

- OpenList는 어떠한 서드파티 파생 프로젝트와도 공식적인 연관이 없습니다.

- 이 프로젝트의 모든 소프트웨어, 코드, 서비스는 OpenList 팀이 유지 관리하며 GitHub에서 무료로 제공됩니다.

- 프로젝트 문서 및 API 서비스는 주로 Cloudflare가 제공하는 무상 리소스에 의존합니다. 현재 유료 플랜이나 상업적 배포는 없으며, 기존 기능 사용에는 어떠한 비용도 들지 않습니다.

커뮤니티의 자유로운 사용과 파생 개발 권리를 존중하지만, 다운스트림 프로젝트에 다음 사항을 강력히 권고합니다:

- 사칭 홍보나 상업적 이익을 위해 "OpenList" 명칭을 사용하지 마십시오.

- OpenList 기반 코드를 비공개 방식으로 배포하거나 AGPL 라이선스 조건을 위반하지 마십시오.

건강한 생태계 발전을 위해 다음을 권장합니다:

- 오픈소스 정신에 따라 프로젝트 출처를 명확히 밝히고 적절한 오픈소스 라이선스를 선택하십시오.

- 상업적 이용 시 프로젝트 이름으로 "OpenList" 또는 혼동을 야기할 수 있는 명칭 사용을 피하십시오.

- OpenListTeam/Logo에 있는 자료를 사용하려면 라이선스 조건에 따라 수정 및 사용하십시오.

OpenList 프로젝트에 대한 지지와 이해에 감사드립니다.
